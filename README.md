# 💌 Projeto "Minha Gatinha" - Carta Virtual

Uma aplicação web delicada e interativa desenvolvida como uma **surpresa especial para minha namorada**. O projeto foi criado para entregar uma mensagem personalizada de forma criativa, transformando uma declaração em uma experiência digital lúdica.

## 💝 O Propósito

Este projeto nasceu do desejo de fazer uma surpresa única. Diferente de uma carta de papel, esta versão digital utiliza interatividade e animações para guiar a leitura, tornando o momento de ler a mensagem ainda mais marcante e envolvente.

## 🚀 Demonstração

**[Veja a surpresa online aqui!](https://spi-alex.github.io/p-ela/)**

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as tecnologias fundamentais da web para garantir que a surpresa funcione perfeitamente em qualquer dispositivo:

- **HTML5**: Estruturação semântica do texto da carta.
- **CSS3**: Estilização romântica com tons de rosa, roxo e branco pastel.
- **JavaScript (Vanilla)**: Lógica para transição das telas e interação com o personagem "gatobruxo".

## 📱 Funcionalidades

- **Interação Inicial**: Um convite lúdico pedindo para clicar no personagem para abrir a surpresa.
- **Mensagem Personalizada**: Um card centralizado com uma mensagem extensa, formatada com carinho para facilitar a leitura no navegador.
- **Guia Visual**: O uso do mascote "gatobruxo" como o condutor da experiência interativa.

## ⚙️ Como Executar o Projeto

Por ser uma página estática, basta abrir o arquivo `index.html` em seu navegador para ver a surpresa em ação.
